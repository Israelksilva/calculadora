# 🧮 Calculadora Científica

Uma calculadora web moderna com interface elegante e funcionalidades matemáticas avançadas.

## 🚀 Funcionalidades
- Operações básicas (+, -, ×, ÷)
- Funções trigonométricas (sin, cos, tan)
- Logaritmo
- Raiz quadrada
- Potência
- Fatorial
- Constantes (π, e)
- Interface responsiva

## 💻 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
